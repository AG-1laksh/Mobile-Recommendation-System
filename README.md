# Samsung Mobile Recommendation System

A starter workspace for building a Samsung product recommendation pipeline.

## Project structure

- `data/` — raw, interim, processed data, and saved models
- `notebooks/` — exploratory analysis and model development notebooks
- `src/` — reusable preprocessing, training, and prediction code
- `ui/` — optional user interface assets
- `reports/` — exported figures, metrics, and evaluation reports
- `images/` — diagrams and visual assets
- `docs/` — project documentation
- `tests/` — automated tests

## Next steps

1. Add your data to `data/raw/`
2. Build preprocessing in `src/preprocess.py`
3. Train models in `src/train_model.py`
4. Track experiments and results in `notebooks/`
